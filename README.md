<h1 align="center"> 🎬 Projeto API [FIlmes] 🎬 </h1>

# :hammer: ferramentas que usamos no projeto

- `Ferramenta 2 - API`: Utilizamos API para facilitar a navegação e o cadastro do filmes e gêneros
